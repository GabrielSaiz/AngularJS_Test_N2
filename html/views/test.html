Introduction

You are about to start performing the technical test that Ricoh proposes for the recruitment of Angularjs developers.

This test consists of 4 independent tests consisting of the development of a table, the development of a form, the development of a summary panel and implement multilanguage. These four tests are thoroughly explained in the following tabs.

The quality and result of this test will have an important weight of decision that the group of evaluators take when evaluating your candidacy to be part of our team.

We are aware that the test can be difficult to face depending on the personal situation of the evaluated, since it is extensive and will take much of your free time. But the compensation will be directly proportional to its result, so we ask for an extra effort to the evaluated that will be rewarded.

Finally, in case of any doubt in the realization of this test, get in touch with our team and we will solve it as quickly as possible.

Sincerely, Sergio Cernuda Fdez angularJs developer on Ricoh.

Test 1

Make a Form

Make a form that simulates inserting an object in the json hosted in "/resources/data/data.json".

Create all the necessary fields in the form for the simulation and treat them correctly from the controller.

After this you have two options

  • Show the object on the same form screen
  • Show the object by console

The form will be inserted in a new tab that you will have to create in the side menu with the correct configuration and using the ui-router library for navigation.

Test 2

1.a Make a table with desktop computers.

Make a table of desktop computers with the information attached to the Json with path "/resources/data/data.json". The table will be inserted in a new tab that you will have to create in the side menu with the correct configuration and using the ui-router library for navigation.

1.b Make a carousel with laptop computers.

Make a carousel of laptop computers with the information attached to the Json with path "/resources/data/data.json". The carousel will be inserted in the same tab of the table.
